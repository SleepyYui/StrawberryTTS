# StrawberryTTS
### StyleTTS 2 fork wit a Gradio interface slapped on top

## Pre-requisites
1. Python >= 3.7
2. Clone this repository:
```bash
git clone https://github.com/SleepyYui/StrawberryTTS.git
cd StrawberryTTS
```
3. Run the installation script:
```bash
./install.sh
```
Or on Windows:
```bash
./install.bat
```

## Usage
1. Run the web interface:
```bash
./run_web.sh
```
Or on Windows:
```bash
./run_web.bat
```
